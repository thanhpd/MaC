== Installation ==

Installing the plugins is just like installing other WordPress plugins. If you don't know how to install plugins, please review the two options below:

Install from ZIP file:

    1. From your Wordpress deshboard, go to Plugins > Add New.
    2. Select 'Upload' from the set of links at the top of the page (the second link)
    3. From here, browse for the ZIP file included in your plugin purchase titled 'dewdrop-scrollbar-wp-free.zip' and click the 'Install Now' button
    4. Onse installation is complete, active the plugin to enable its features.

Install by FTP:

    1. Find the directory titled 'dewdrop-scrollbar' and upload it and all files within to the plugins directory of your WordPress instal (/WORDPRESS-DIRECTORY/wp-content/plugins/) [e.g.: www.yourdomain.com/wp-content/plugins/]
    2. From your WordPress dashboard, chose 'Installed Plugins' option under the 'Plugins' category
    3. Locate the newly added plugin and click on the 'Activate' link to enable its features.
